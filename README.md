# PIC18F47Q10-With-Mikroe-Temp-Hum-3
Display Temperature & Humidity Using Data Visualizer

Hardware Used: PIC18F47Q10 Curiosity Nano PIC MCU Board, Curiosity Nano Base for Click Boards, and MIKROE Temp&Hum 3 board. The MIKROE board is plugged into mikro BUS 1 of the Nano base board.

Software Used: Microchip MPLAB X IDE, MCC Application Builder, and Data Visualizer. I2C reference document is Microchip TB3281.

After the hardware is setup, download the program, then open the Data Visualizer. Set the Data Visualizer COM port and Baud Rate to 9600. 

Other functions can easily be added as needed.
